## Content

本期《AI 炼金术》探讨了 AI Agent 的发展与教育的未来。嘉宾李可佳分享了从 AI 辅助阅读产品 Ibrary 到 AI 替代学习 BotLearn 的转变，核心在于 OpenCore 的出现，它为 Agent 提供了自主性和持久记忆。BotLearn 旨在解决 Agent 早期技能不足的问题，通过 KOL 推荐和社区协议来保证 skill 的质量与安全。讨论还涉及了 Agent 产品与人类产品的差异，Agent 更需要原子化和可组合性，以及对协议位置的争夺。李可佳认为，未来教育稀缺的是新的学习关系，人需要学习判断、视野和价值观，并对自己的决定承担责任。

Takeaways:
1. 智能体并不需要人类产品所必需的界面，它们更需要高效的协议、更灵活的组合以及及时的反馈。
2. 未来社会可能会奖励一部分眼高手低的人，因为他们可以利用 AI 工具实现想法，即使缺乏执行细节的耐心。
3. 学习并非天生的刚需，社会真实反馈表明，为他人提供学习机会的市场远大于自学市场。
4. 随着大模型的理解和编排能力提升，那些能够被明确描述结果的技能，可能会通过工作流直接交付，并进行交易。
5. 针对 Agent 的产品设计应抛弃 “Don't Make Me Think” 的思路，转向能力和资源的原子化，并根据场景灵活组合。
6. 巡逻虾、安全公约以及奖励函数等社区协议，能够保证真实用户的使用经验和进化故事在社区内分享。
7. 社区可以更好地满足用户在使用 Agent 过程中遇到的各种 corner case 需求。
8. 传统商业的壁垒在于切换 CRM 时的摩擦成本，但 Agent 可以在不同平台间自由切换，因此协议层至关重要。
9. 2026 年可能是最后一年还大量依赖人与人之间的摩擦创造价值的时光，此后创业者的价值可能会不断被 AI 替代。
10. Agent 社区的重点在于建立良好的氛围，让 Agent 主动找到并融入，而非强制推广。
